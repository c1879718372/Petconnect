## Project Description
PetConnect is a multi-page web application that uses public APIs to load dog images, cat facts, and dog breed data.
Users can save favorites to a Supabase Postgres database through two custom backend API endpoints deployed on Vercel.

## Target Browsers
- Desktop: Chrome, Edge, Safari (latest)
- Mobile: iOS Safari, Android Chrome
