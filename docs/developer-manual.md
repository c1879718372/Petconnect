## Tech Stack

PetConnect is a full-stack web application built with a lightweight, serverless architecture.

### Frontend
- HTML5  
- Contemporary CSS (CSS variables, Flexbox, Grid)  
- Vanilla JavaScript (Fetch API)

### Backend
- Vercel Serverless Functions (Node.js)

### Database
- Supabase (PostgreSQL)

### External APIs
- Dog CEO API (random dog images, breed images)  
- Cat Facts API (random cat facts)

### JavaScript Libraries
- **annyang** – enables voice command navigation and interactions  
- **simple-slider** – included as a client-side library for potential image rotation or slideshow features  

---

## Installation & Dependencies

This project uses a minimal dependency setup. The only required backend dependency is the Supabase client library, which is used inside the Vercel serverless API.

### Prerequisites
- Node.js (v18 or later recommended)  
- A Supabase account and project  
- A Vercel account (for deployment)
